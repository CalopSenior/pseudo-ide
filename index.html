O que é a Pseudo-IDE?

                    

                        A Pseudo-IDE é um ambiente de programação educacional onde você escreve pseudocódigo em
                            português
                        e ele é automaticamente executado no navegador. O objetivo é aprender lógica de programação,
                        matemática e
                        algoritmos sem precisar decorar sintaxe de linguagens complexas.
                    


                    
Estrutura obrigatória de todo algoritmo

                    

                        Todo programa precisa de um nome de algoritmo seguido por () e um
                        bloco { }.
                        Pense nisso como a "capa" do seu programa.
                    

                    
// O nome do algoritmo aparece na aba e no topo da janela.
// Pode ser qualquer nome sem espaços.
meuPrimeiroAlgoritmo()
{
    imprima("Olá, Mundo!");
}


                    
Entrada e Saída

                    

                        
                            

                                Comando
                                O que faz
                                Exemplo
                            

                        
                        
                            

                                imprima(valor)
                                Mostra um valor no console de saída. Aceita qualquer coisa: números, textos, listas…
                                
                                imprima("olá")
                            

                            

                                leia("pergunta")
                                Pausa o programa e espera o usuário digitar algo. O valor digitado fica armazenado
                                    na variável.
                                caracter n = leia("Seu nome:")

   ou para desativar vários comandos de uma vez.

Seu primeiro programa completo

                    
saudacao()
{
    caracter nome = leia("Qual é o seu nome?");
    inteiro idade = leia("Quantos anos você tem?");

        imprima("Você é maior de idade.");
    } senao {

                        com o número aproximado da linha onde aconteceu.
                    

                



                
                


                    
Tipos de variável

                    

                        Uma variável é uma "caixa" onde você guarda um valor. Na Pseudo-IDE, você declara o
                            tipo
                        da caixa antes do nome — isso ajuda a evitar erros: se você tentar guardar um texto em uma caixa
                        de inteiro,
                        o programa avisa na hora.
                    

                    

                        
                            

                                Tipo
                                Guarda
                                Exemplo
                                Observação
                            

                        
                        
                            

                                inteiro
                                Números sem casas decimais
                                inteiro x = 10
                                Aceita negativos. Não aceita 3.5, somente 3.
                            

                            

                                real
                                Números com ou sem casas decimais
                                real preco = 9.99
                                Use ponto, não vírgula: 3.14 (não 3,14).
                            

                            

                                caracter
                                Textos (qualquer sequência de letras, números, símbolos)
                                caracter nome = "Ana"
                                Coloque o texto entre aspas duplas ou simples.
                            

                            

                                booleano
                                Verdadeiro ou falso
                                booleano ativo = verdadeiro
                                Só aceita verdadeiro ou falso.
                            

                            

                                super
                                Qualquer coisa (tipo universal)
                                super lista = [1, 2, 3]
                                Use quando precisar de listas com [...] ou misturar tipos. Sem
                                    validação de tipo.
                            

                        
                    


                    

                        Tipagem forte: a IDE verifica o tipo no momento da atribuição.
                        inteiro x = 3.7 gera erro imediatamente: "variável 'x' declarada como inteiro,
                            mas recebeu real (3.7)".
                        Isso te ajuda a encontrar bugs antes que eles causem problemas maiores.
                    


                    
Valores especiais

                    

                        
                            

                                Palavra
                                Significado
                                Quando usar
                            

                        
                        
                            

                                verdadeiro
                                Booleano verdadeiro (true)
                                Condições, flags de controle
                            

                            

                                falso
                                Booleano falso (false)
                                Condições, flags de controle
                            

                            

                                vazio
                                Ausência intencional de valor (null)
                                Inicializar variável "sem valor ainda"
                            

                            

                                indefinido
                                Variável ainda não atribuída (undefined)
                                Checar se algo foi preenchido
                            

                            

                                Infinito
                                +∞
                                Valores que crescem sem limite, divisão por zero
                            

                            

                                NegInfinito
                                −∞
                                Limite negativo

Textos especiais

                    
F-strings — texto com variáveis incorporadas

                    
Coloque f antes das aspas e use { } para incluir
                        variáveis dentro do texto.

                    
caracter nome = "Carlos";
inteiro idade = 25;
imprima(f"Meu nome é {nome} e tenho {idade} anos.");
// Saída: Meu nome é Carlos e tenho 25 anos.

// Você pode usar expressões dentro das chaves:
imprima(f"O dobro de {idade} é {idade * 2}.");
// Saída: O dobro de 25 é 50.


                    
Strings brutas (raw strings) — prefixo #

                    

                        Use #"..." ou #'...' quando a string contém \ que não devem ser interpretados como escape do JavaScript.
                        Isso é essencial para escrever comandos LaTeX sem precisar dobrar cada barra:
                    

                    
importar latex como lt;

// Com raw string: funciona direto
lt.linha(#"\frac{1}{2} + \frac{1}{3} = \frac{5}{6}")
lt.bloco(#"\sum_{k=1}^{n} k = \frac{n(n+1)}{2}")
lt.linha(#"e^{i\pi} + 1 = 0")

// Sem raw string: \f vira form feed, \p vira p — LaTeX corrompido
lt.linha("\frac{1}{2}") // ERRADO — \f é interpretado pelo JS

                    

                        Quando usar: sempre que a string contiver \ — para LaTeX, expressões regulares literais ou qualquer texto com barras.

                        Limitação: não é possível incluir o mesmo caractere de aspas dentro do raw string (use #'...' se precisar de " no conteúdo).
                    


                    
Repetição de texto

                    
imprima(3 * "ha");    // hahaha
imprima("=-" * 10); // =-=-=-=-=-=-=-=-=-=-=-=-


                    
Notação científica

                    
real grande = 2E10;    // 20 000 000 000
real pequeno = 1.5e-3; // 0.0015


                    
Listas fixas com super

                    

                        Use super e colchetes [...] para criar uma lista de acesso rápido por
                        índice.
                        O índice começa em 0 (o primeiro elemento é o de número 0).
                    

                    
super dias = ["Seg", "Ter", "Qua", "Qui", "Sex"];

imprima(dias[0]);  // Seg  (primeiro)
imprima(dias[4]);  // Sex  (quinto)
imprima(dias.length); // 5 (total de elementos)

                



                
                


                    
Condicional: se / senao

                    
Execute um bloco de código somente se uma condição for verdadeira.

                    
// Forma simples
se (nota >= 7) {
    imprima("Aprovado!");
}

// Com alternativa
se (nota >= 7) {
    imprima("Aprovado!");
} senao {
    imprima("Reprovado.");
}

// Múltiplas condições (encadeado)
se (nota >= 9) {
    imprima("Excelente!");
} senao se (nota >= 7) {
    imprima("Bom");
} senao se (nota >= 5) {
    imprima("Regular");
} senao {
    imprima("Insuficiente");
}


                    
Escolha (switch)

                    
Ideal quando você quer comparar uma variável com vários valores fixos. Mais
                        legível que vários senao se.

                    
inteiro dia = 3;
escolha (dia) {
    caso 1: imprima("Segunda-feira"); quebrar;
    caso 2: imprima("Terça-feira");  quebrar;
    caso 3: imprima("Quarta-feira"); quebrar;
    padrao:  imprima("Outro dia"); // executado se nenhum caso bater
}
// Saída: Quarta-feira

                    
⚠️ Atenção: não esqueça o quebrar ao final de
                        cada caso. Sem ele, o programa continua executando os casos seguintes.


                    
Laço: para (contagem)

                    
Repete um bloco um número determinado de vezes. Perfeito quando você sabe
                        exatamente quantas repetições quer.

                    
// Estrutura: para (inicio; condição; passo)
para (inteiro i = 1; i <= 5; i++) {
    imprima(f"Iteração número {i}");
}
// Imprime: 1, 2, 3, 4, 5

// Contagem regressiva
para (inteiro i = 10; i >= 1; i--) {
    imprima(i);
}

// Passo de 2 em 2
para (inteiro i = 0; i <= 10; i = i + 2) {
    imprima(i); // 0, 2, 4, 6, 8, 10
}


                    
Laço: para em (percorrer coleção)

                    
Percorre cada elemento de uma Lista, Conjunto, Mapa ou texto, um por vez. Mais
                        simples que o laço com contador quando você não precisa do índice.

                    
importar metodos como m;
super frutas = m.lista("maçã", "banana", "uva");

para (fruta em frutas) {
    imprima(fruta);
}
// Imprime: maçã, banana, uva (um por linha)

// Também funciona em textos (itera caractere a caractere)
para (letra em "olá") {
    imprima(letra); // o, l, á
}

// E em Conjuntos
super nums = m.conjunto();
nums.adicionar(10); nums.adicionar(20); nums.adicionar(30);
para (n em nums) { imprima(n); }


                    
Laço: enquanto

                    
Repete enquanto uma condição for verdadeira. Use quando você não sabe
                        de antemão quantas repetições vai precisar.

                    
inteiro tentativas = 0;
inteiro senha = 1234;
inteiro entrada = 0;

enquanto (entrada != senha) {
    entrada = leia("Digite a senha:");
    tentativas = tentativas + 1;
}
imprima(f"Correto! {tentativas} tentativa(s).");


                    
Laço: faca...enquanto

                    
Igual ao enquanto, mas sempre executa pelo menos uma
                            vez antes de checar a condição.

                    
inteiro n = 0;
faca {
    n = leia("Digite um número positivo:");
} enquanto (n <= 0);
imprima(f"Você digitou: {n}");


                    
Controle de laços: quebrar e continuar

                    
// quebrar — sai do laço imediatamente
para (inteiro i = 1; i <= 10; i++) {
    se (i == 5) { quebrar; }
    imprima(i); // imprime 1, 2, 3, 4
}

// continuar — pula para a próxima iteração
para (inteiro i = 1; i <= 5; i++) {
    se (i == 3) { continuar; } // pula o 3
    imprima(i); // imprime 1, 2, 4, 5
}

                



                
                


                    
O que é uma função?

                    

                        Uma função é um bloco de código com um nome, que você pode chamar quando precisar.
                        Use funções para evitar repetição e organizar seu algoritmo em
                        partes menores.
                    


                    
Função simples

                    
calculadora()
{
    // Definir a função
    funcao somar(a, b) {
        retorno a + b;
    }

    // Chamar a função
    imprima(somar(3, 4));    // 7
    imprima(somar(10, -2));   // 8

    // Guardar o resultado em variável
    real resultado = somar(5.5, 2.5);
    imprima(resultado); // 8
}


                    
Funções com tipo nos parâmetros (tipadas)

                    

                        Você pode declarar o tipo esperado de cada parâmetro usando a sintaxe
                        tipo(nome). Se o chamador passar um tipo errado, um erro claro é gerado
                        automaticamente.
                    

                    
funcao dobrar(real(x)) {
    retorno x * 2;
}

imprima(dobrar(5));     // 10
imprima(dobrar(3.14));  // 6.28
dobrar("oi");           // ERRO: esperava real, recebeu texto


                    
Parâmetros com restrição de valor

                    

                        Além do tipo, você pode adicionar uma condição que o valor deve satisfazer.
                        Coloque a condição dentro dos parênteses do tipo.
                    

                    
funcao raizSegura(real(x >= 0)) {
    retorno raiz(x);
}

imprima(raizSegura(9));   // 3
raizSegura(-1);          // ERRO: 'x' deve satisfazer: x >= 0

// Outros exemplos de restrições:
funcao dividir(real(a), real(b != 0)) {
    retorno a / b;
}
funcao fatPosit(inteiro(n >= 0)) { /* ... */ }


                    
Parâmetros com valor padrão

                    

                        Use : valor após o parêntese para definir um valor padrão.
                        Parâmetros com padrão podem ser omitidos na chamada.
                    

                    
funcao saudar(caracter(nome), caracter(prefixo) : "Olá") {
    imprima(f"{prefixo}, {nome}!");
}

saudar("Ana");             // Olá, Ana!
saudar("Bruno", "Oi");    // Oi, Bruno!

// Combinando tipo + restrição + padrão:
funcao potencia(real(base), inteiro(exp >= 1) : 2) {
    retorno expo(base, exp);
}
imprima(potencia(3));      // 9  (exp usa padrão 2)
imprima(potencia(2, 10)); // 1024


                    
Recursão

                    
Uma função pode chamar a si mesma. Sempre defina uma condição de
                            parada para evitar loop infinito.

                    
funcao fatorial(inteiro(n >= 0)) {
    se (n <= 1) { retorno 1; }    // condição de parada
    retorno n * fatorial(n - 1); // chamada recursiva
}
imprima(fatorial(5)); // 120  (5 × 4 × 3 × 2 × 1)
imprima(fatorial(0)); // 1   (por definição, 0! = 1)


                    

                        Resumo das sintaxes de parâmetro:

                        tipo(nome) — obrigatório, sem restrição

                        tipo(nome op valor) — obrigatório, com restrição (ex: >= 0,
                        != 0)

                        tipo(nome) : padrão — opcional com valor padrão

                        tipo(nome op valor) : padrão — opcional com restrição e padrão
                    

                



                
                


                    
O que são exceções?

                    

                        Exceções são erros que acontecem durante a execução do programa. Você pode
                        lançar uma exceção para sinalizar que algo deu errado, e
                        capturá-la
                        para tratar o problema de forma elegante — sem que o programa trave.
                    


                    
Capturar erros: tentar / capturar

                    
Envolva código arriscado em tentar { } e trate o erro em
                        capturar { }.
                    

                    
tentar {
    real resultado = 10 / 0;
    imprima(resultado);
} capturar (erro) {
    imprima(f"Aconteceu um erro: {erro.message}");
}

// O programa continua após o capturar:
imprima("Programa continua normalmente.");


                    
Lançar exceção simples

                    
funcao sacar(real(valor > 0)) {
    real saldo = 100.0;
    se (valor > saldo) {
        lançar erro "Saldo insuficiente para o saque.";
    }
    imprima(f"Saque de R${valor} realizado.");
}

tentar {
    sacar(50);   // OK
    sacar(200);  // lança erro
} capturar (e) {
    imprima(e.message); // Saldo insuficiente para o saque.
}


                    
Lançar exceção estruturada

                    

                        Use lançar erro(tipo, mensagem, dadoRecebido, dadoEsperado) para criar erros mais
                        ricos,
                        com informações detalhadas sobre o que deu errado. Todos os parâmetros são opcionais.
                    

                    
funcao validarIdade(inteiro(n)) {
    se (n < 0 ou n > 150) {
        lançar erro(
            "ValorInvalido",      // tipo do erro
            "Idade fora do intervalo válido", // descrição
            n,                      // dado recebido
            "entre 0 e 150"        // dado esperado
        );
    }
    imprima(f"Idade {n} registrada.");
}

tentar {
    validarIdade(25);  // OK
    validarIdade(999); // lança erro estruturado
} capturar (e) {
    imprima(f"[{e.tipo}] {e.message}");
    imprima(f"Recebido: {e.dadoEntrada}");
    imprima(f"Esperado: {e.dadoEsperado}");
}


                    

                        
                            

                                Forma
                                Sintaxe
                                Quando usar
                            

                        
                        
                            

                                Simples
                                lançar erro "mensagem"
                                Sinalização rápida de problema
                            

                            

                                Estruturada
                                lançar erro(tipo, msg, entrada, esperado)
                                Erros ricos com contexto para diagnóstico
                            

                            

                                Qualquer valor
                                lançar expressao
                                Lançar um objeto ou string personalizado

Aleatoriedade e somatório

                    

                        
                            

                                Função
                                Parâmetros
                                Descrição
                            

                        
                        
                            

                                m.aleatorio(min, max)
                                min, max: inteiros
                                Inteiro aleatório no intervalo [min, max] (inclusivo em ambos os lados).
                            

                            

                                m.somatorio(fn, min, max)
                                fn: funcao(i), min/max: numeros
                                Calcula Σ fn(i) para i de min até max. Substitui a notação matemática de somatório.
                                
                            

                            

                                m.produtorio(fn, min, max)
                                fn: funcao(i), min/max: numeros
                                Calcula Π fn(i) para i de min até max.
                            

                        
                    

                    
importar mat como m;

// Somatório: Σ i² de 1 a 5 = 55
funcao quadrado(i) { retorno i * i; }
imprima(m.somatorio(quadrado, 1, 5)); // 55

// Fatorial via produtório: Π i de 1 a 5 = 120
funcao id(i) { retorno i; }
imprima(m.produtorio(id, 1, 5)); // 120

Texto — m.caracter(valor)

                    
Cria um objeto de texto com métodos de manipulação. Retorna um novo Caracter
                        sem modificar o original.

                    
importar metodos como m;
super t = m.caracter("  olá mundo  ");

imprima(t.maiusculo());     // "  OLÁ MUNDO  "
imprima(t.minusculo());     // "  olá mundo  "

Editor .pdados

                    
No Importador de Dados, use a aba Editor .pdados para criar arquivos do zero. Cada coluna pode ser definida por:

                    
    
                      
  • expressão — qualquer expressão de biblioteca: mat.linspace(0, 2*mat.PI, 100), x.map(v => mat.sen(v)), etc.
    • 
                      
  • manual — valores digitados linha a linha ou separados por vírgula.
    • 
                    

                    
Colunas definidas anteriormente ficam disponíveis como variáveis nas expressões das colunas seguintes (ex: defina x e use x.map(…) em y). Ao salvar, os dados calculados são gravados no .pdados; o código gerado usa as expressões originais para facilitar reedição.

                    
// Coluna x: mat.linspace(0, 2*mat.PI, 60)
// Coluna y: x.map(v => mat.sen(v))
// Coluna z: x.map(v => mat.cos(v))

// Código gerado (preserva as expressões):
importar mat como mat;

super x = mat.linspace(0, 2*mat.PI, 60);
super y = x.map(v => mat.sen(v));
super z = x.map(v => mat.cos(v));

Área do Professor: Criação de Bibliotecas e API

                
            


            

                

                    A Pseudo-IDE permite criar, empacotar e distribuir suas próprias bibliotecas de funções para os
                    alunos usando um servidor em nuvem (API REST).
                


                
1. A Anatomia de uma Biblioteca

                
Uma biblioteca é um algoritmo normal que contém funções e um cabeçalho de metadados. O marcador
                    principal é o @BIBLIOTECA[nome_do_pacote].


                
    
                    
  • Tudo deve ser Função: Códigos soltos (como imprima()) fora de
                        funções serão executados assim que o aluno importar o pacote. Coloque sua lógica de negócio
                        dentro de blocos funcao.
    • 
                    
  • Funções Públicas: Qualquer função declarada normalmente (ex:
                        funcao calcular()) ficará disponível para os alunos usarem.
                    
    • 
                    
  • Funções Privadas: Se quiser criar uma função auxiliar oculta para uso interno,
                        inicie o nome com um sublinhado (ex: funcao _ajuste()). Ela não será exportada para
                        o usuário.
    • 
                


                
2. Exemplo Completo de Pacote

                
@NOME[Física Básica]
@AUTOR[Prof. Gabriel]
@VERSAO[1.0]
@BIBLIOTECA[fisica_basica]

minhaBiblioteca() {
    importar mat como m;

    // Função Privada (Oculta do aluno)
    funcao _validarTempo(t) {
        se (t < 0) { lançar erro "Tempo não pode ser negativo!"; }
    }

    // Função Pública
    funcao mru(pos_inicial, vel, tempo) {
        _validarTempo(tempo);
        retorno pos_inicial + (vel * tempo);
    }
}


                
3. Como Configurar o Servidor (Backend)

                
A Pseudo-IDE não exige um banco de dados complexo. O servidor só precisa aceitar requisições
                    POST (texto simples) para salvar o pacote e GET para retornar o
                    pacote em JSON.
                


                

                    Por que texto simples no POST? O envio é feito via Simple Request
                    (text/plain) para evitar bloqueios automáticos de CORS (requisições OPTIONS de
                    preflight) em servidores serverless (ex: Google Apps Script).
                


                
Exemplo de Servidor: Google Apps Script (Planilha)

                
// doPost: Grava a biblioteca na Planilha
function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  const data = JSON.parse(e.postData.contents);
  sheet.appendRow([data.nome, data.autor, data.versao, data.codigoInjetor]);
  return ContentService.createTextOutput("OK").setMimeType(ContentService.MimeType.TEXT);
}

// doGet: Busca a biblioteca mais recente (lendo de baixo para cima)
function doGet(e) {
  const nome = e.parameter.nome;
  const data = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet().getDataRange().getValues();
  for (let i = data.length - 1; i >= 0; i--) {
    if (data[i][0] === nome) {
      const payload = { tipo: 'biblioteca', nome: data[i][0], autor: data[i][1], versao: data[i][2], codigoInjetor: data[i][3] };
      return ContentService.createTextOutput(JSON.stringify(payload)).setMimeType(ContentService.MimeType.JSON);
    }
  }
  return ContentService.createTextOutput("{}").setMimeType(ContentService.MimeType.JSON);
}


                
4. Como o Aluno Utiliza

                
No cabeçalho da Pseudo-IDE, o aluno clica no botão ⚙️ API e cola a URL do seu
                    servidor. Depois, basta importar o pacote:

                
importar fisica_basica como fb;

imprima(fb.mru(0, 10, 5)); // Resultado: 50
