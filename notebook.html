Guia do Notebook
Visão Geral

O Pseudo Notebook organiza código e texto em células intercaladas. Células de código executam algoritmos em pseudo-linguagem; células de texto aceitam Markdown com LaTeX para documentação.

Arquivos são salvos no formato .pseudonb (JSON) e podem ser exportados como HTML estático.

Células de Código
  • Ctrl+Enter — executar a célula atual
  • Ctrl+Shift+Enter — executar tudo
  • Tab — indentar (2 espaços)
  • Botão ▶ Executar na toolbar da célula

A saída aparece abaixo da célula. Erros são exibidos em vermelho com número da linha.

// Exemplo básico
inteiro n = 10
real soma = 0

para (inteiro i = 1; i <= n; i++) {
    soma = soma + i
}
imprima(f"Soma de 1 a {n}: {soma}")
Modo Ambiente (amb)

Ative o botão amb na toolbar de uma célula de código para expor suas variáveis ao ambiente compartilhado. Variáveis que começam com _ não são exportadas.

Ao executar individualmente qualquer célula, o ambiente é injetado automaticamente — você pode ler variáveis de células amb mesmo sem executar tudo.

amb 
// Célula 1 — configura parâmetros (amb ativo)
inteiro periodos = 12
real taxa = 0.015

funcao montante(p, i, n) {
    retorne p * (1 + i) ^ n
}
sem amb 
// Célula 2 — usa o ambiente da célula 1
real resultado = montante(1000, taxa, periodos)
imprima(f"Montante: {resultado:.2f}")
Modos de Executar Tudo

Clique no ▼ ao lado de Executar Tudo para escolher o modo:

ModoComportamento
Amb ★ (padrão) Cada célula roda isolada com o ambiente injetado. Somente células amb escrevem no ambiente. Ideal para notebooks com células de "configuração" e células de "uso".
Livre Todas as células rodam em um único escopo — qualquer variável é automaticamente visível nas células seguintes, independente do modo amb.

ⓘ Em ambos os modos, ao final da execução as variáveis das células amb são capturadas para o ambiente, permitindo execuções individuais subsequentes.

LaTeX — Delimitadores e Blocos

Renderizado via KaTeX. Quatro formas de inserir fórmulas:

SintaxeTipoExemplo
$...$InlineA área é $\pi r^2$.
\(...\)Inline (alternativo)\(E = mc^2\)
$$...$$Bloco centralizado$$\int_0^\infty e^{-x}dx$$
\[...\]Bloco centralizado (alternativo)\[\frac{a}{b} = c\]

Os delimitadores $$ e \[ aceitam múltiplas linhas:

$$
\int_0^1 f(x)\,dx \approx \sum_{i=0}^{n} f(x_i)\,\Delta x
$$

Ambientes \begin{...}...\end{...} são detectados automaticamente como blocos:

\begin{align}
  f(x) &= ax^2 + bx + c \\
  f'(x) &= 2ax + b
\end{align}

Ambientes suportados: equation, align, aligned, gather, multline, split, cases, pmatrix, bmatrix, vmatrix, matrix, array.

ⓘ Use \$ para um cifrão literal dentro de texto.
